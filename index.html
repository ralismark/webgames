Welcome to ralismark webgames!

This is a collection of fun web things that I've made over the past few years. Pretty much all of these were from when I was new to programming (Javascript being my first language), and so there's a conspicuous lack of good programming practices.

  1. pong.html, a pong game with many options and local multiplayer support for 2 or 4 people!
  2. ship.html, a singleplayer ship game inspired by Astro Party. Controls are
    • S to turn
    • A to shoot
    • Z to pause
    • X to resume
    I never got around to coding up a config UI, but all the ships are specified in the pl variable and by editing it it's also possible to have multiple players (e.g. add p2 to it).
  3. Basalt Bashers, the web port of my project for the HSC Software Design and Development subject. Controls are clicking the green squares. The original was in pretty awful C++ and is available on github if you want to laugh look at it. This port was made quite a bit later than the other two projects.
  4. Gravity Testing, a simulation of n-body gravity.
    • Clicking moves the spaceship (the red rectangle) while preserving velocity
    • Space prompts for new position/velocity values
    • 1 stops the simulation
    • Z starts simulation, and increases speed for each subsequent press
    • C clears the trail
    • X does something related to the trail, but I can't tell what
    • W/A/S/D boosts in those directions

These use no frameworks (except for jQuery), have no build process or minifier, and just the DOM/Canvas API. They're very low-polish and are built around being able to tweak the javascript code. Everything is contained in the handwritten .js files and available on Github.
